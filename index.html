How to Print this Map

To print or save this as an image, first set up the map the way you want it displayed and then press Ctrl-H to hide the admin interface.

You may now use an image capture, such as Awesome Screenshot or simply Print Screen (using Ctrl-Alt-PrtScr to grab the current window -- for best results, make your browser full screen).
